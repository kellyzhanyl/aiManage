git 
1.下载git工具 地址为：http://git-scm.com/downloads

2.安装【省略】

3.配置用户信息
  git config --global user.name "xxxx"
  git config --global user.email xxx@xxx.com
  
4.查看配置信息
  git config --list
 
5.克隆git代码库
  git clone https://github.com/kellyzhanyl/aiManage.git
  
6.创建本地分支
  git branch master
  git branch dev
 
7.添加本地文件
  git add .

8.提交文件至缓存
  git commit -m "描述语"
  
9.提交上传
  git push  [如果远程仓库与本地仓库名不同或者不存在,可以使用以下] git push -u origin master/release[将本地仓库名同步到远程仓库]
